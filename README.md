## 📋 Description

Stage de recherche à l'**Université Alexandru Ioan Cuza (UAIC)** de Iași en Roumanie. Ce projet porte sur deux axes complémentaires : l'**anonymisation de données médicales sensibles** (techniques de k-anonymat et pseudonymisation) et la **modélisation prédictive par régression logistique** pour la classification de données de santé.

## 🎯 Objectifs du stage
- Étudier et appliquer les techniques d'anonymisation des données personnelles (RGPD)
- Implémenter le k-anonymat et la pseudonymisation sur des données médicales
- Construire et évaluer un modèle de régression logistique (classification binaire)
- Évaluer les performances du modèle (ROC, AUC, matrice de confusion)
- Travailler en environnement de recherche international

## 📊 Résultats / Points clés

- **Anonymisation** : Implémentation du k-anonymat (k=5) et pseudonymisation sur des données médicales
- **Modèle** : Régression logistique avec AUC > 0.85 sur les données de test
- **Conformité RGPD** : Analyse et application des principes de minimisation des données
- **Valorisation** : Rapport de stage et présentation des résultats à l'équipe de recherche UAIC
